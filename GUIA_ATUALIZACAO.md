# 🚀 Atualização PlazaYa — Integração de Funcionalidades ConcursosBrasil

## Resumo das mudanças

O PlazaYa será atualizado para incluir TODAS as funcionalidades do ConcursosBrasil:

### Funcionalidades NOVAS no PlazaYa:
1. **Firebase Integration** — Analytics, Firestore, FCM Push Notifications
2. **Tela de Estudo (StudyHomeScreen + StudyQuizScreen)** — Quiz de estudo/simulado
3. **Tela de Perfil (PerfilScreen)** — Conta do usuário, notificações, configurações
4. **Auth completo** — Cadastro, Login, Reset de senha via Supabase Auth
5. **Push Notifications** — FCM nativo com tópicos por área
6. **Analytics** — Tracking de telas, onboarding, quiz, cliques
7. **UTM Tracking** — Install referrer do Play Store
8. **Modal de Atualização Forçada** — Verifica versão mínima no Supabase
9. **Interstitial Persistente** — Ad a cada 3 minutos de navegação
10. **Tela de Convocatoria (detalhe)** — Abre detalhes da vaga
